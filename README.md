# LiSoft - Site Institucional

Site institucional da LiSoft, empresa especializada em desenvolvimento de sistemas completos.

## Sobre o Projeto

Este é o site oficial da LiSoft, apresentando os serviços da empresa:
- **Sistemas de Pagamento**: Integrações com gateways de pagamento confiáveis
- **Aplicativos**: Desenvolvimento de apps funcionais e intuitivos
- **Sites**: Criação de sites responsivos e modernos

## Estrutura do Projeto

```
web-project/
├── index.html      # Página principal HTML
├── styles.css      # Estilos CSS responsivos
├── script.js       # Funcionalidades JavaScript
└── README.md       # Documentação do projeto
```

## Funcionalidades

- ✅ Design responsivo e moderno
- ✅ Navegação suave entre seções
- ✅ Animações ao scroll
- ✅ Formulário de contato interativo
- ✅ Layout profissional e atraente

## Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Ou use um servidor local para desenvolvimento:
   - **Python**: `python -m http.server 8000`
   - **Node.js**: `npx http-server`
   - **VS Code**: Use a extensão "Live Server"

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e Flexbox/Grid)
- JavaScript (ES6+)
- Design responsivo

## Seções do Site

1. **Hero Section**: Apresentação principal da empresa
2. **Sobre**: Informações sobre a LiSoft
3. **Serviços**: Cards com os principais serviços oferecidos
4. **Contato**: Formulário para entrar em contato

## Personalização

Para personalizar o site, você pode:
- Alterar cores no arquivo `styles.css` (variáveis CSS no `:root`)
- Modificar conteúdo no arquivo `index.html`
- Adicionar novas funcionalidades no arquivo `script.js`
